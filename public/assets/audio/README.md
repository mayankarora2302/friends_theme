# Audio Files Guide

The Sound Box component on each day expects audio files in this directory.

## Required Files

Place your audio files here with these names:
- `day1.mp3` - Rose Day soundtrack
- `day2.mp3` - Propose Day serenade  
- `day3.mp3` - Chocolate Day caf√© vibes
- `day4.mp3` - Teddy Day comfort music
- `day5.mp3` - Promise Day melody
- `day6.mp3` - Hug Day warmth
- `day7.mp3` - Kiss Day romance
- `day8.mp3` - Valentine's Day finale song

## Supported Formats
- MP3 (recommended)
- WAV
- OGG
- M4A

## Optional Sound Effects
- `door-open.mp3` - Login door opening sound
- `error.mp3` - Wrong password sound
- `laugh.mp3` - Sitcom laugh track
- `shutter.mp3` - Camera shutter for photos
- `coffee.mp3` - Coffee cup clink

## Note
If audio files are missing, the Sound Box will still display but won't play audio. This is intentional to prevent errors.
